Tableau Build Guide · How to Recreate Each Chart
🟦
Waffle / Unit Chart
Index calc → square number field. Colour by fate category. Works as a square scatter plot.
🕸️
Radar Chart
Reshape to long format. Use circular layout with path calculations or Radar Extension.
🫧
Bubble Chart
Native in Tableau — recycling_rate on X, target on Y, plastic_use as Size. Colour by sector.
📈
Dual Line + Target
Dual-axis line chart. Actual + projection on one axis, target on second. Reference line for 2030.
📊
Progress Bars
Gantt bar chart. achieved as bar length, target as reference line. Colour by gap %. Sort descending.
